
# Fly Pakistan – GeoFS Virtual Airline

Fly Pakistan is a fictional virtual airline created for the GeoFS flight simulator, based at Jinnah International Airport, Karachi (OPKC). The airline operates a long-haul fleet of Boeing 777-300ER aircraft with Boeing 787-10 Aircraft and a medium-haul fleet of Airbus A320neo, of connecting Pakistan to major international destinations.

This project follows the official GeoFS Virtual Airline guidelines and is compatible with the GeoFS Livery Selector.

## 🎨 Livery Design
- White fuselage
- Green tail inspired by the Pakistan
- Lime engines
- red star on the tail and right next on the text "Fly Pakistan"
